PROJECT TITLE: AirBnB Project
PURPOSE OF PROJECT: Provide GUI for a property search application (real-world data set provided by AirBnB) which allows user to store and view their favourite
properties.
VERSION or DATE: 1.00
HOW TO START THIS PROJECT: Run 'AppView'
AUTHORS: Dennis Kolley
USER INSTRUCTIONS: Select a price range for properties to view. This will allow the user to view all parts of the
application.
